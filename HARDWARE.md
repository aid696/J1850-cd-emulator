## Pinout

| Pin | Function              |
|-----|-----------------------|
| D0  | RX                    |
| D1  | TX                    |
| D2  | Analog Input          |
| D3  | PWM Output            |
| D4  | Digital I/O           |
| D5  | PWM Output            |
| D6  | Digital I/O           |
| D7  | Digital I/O           |
| A0  | Analog Input          |
| A1  | Analog Input          |
| A2  | Analog Input          |
| A3  | Analog Input          |
| A4  | Analog Input (SDA)    |
| A5  | Analog Input (SCL)    |

## Notes
- Ensure correct voltage levels are adhered to when connecting to vehicle diagnostics.
- Double-check connections to avoid short circuits.
